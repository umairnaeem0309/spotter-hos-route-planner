# HOS Route Planner

A Django and React assessment application that creates an HOS-compliant
truck trip plan and generates driver daily log sheets.

## Status

Project foundation is being prepared.

## Planned stack

- Django
- Django REST Framework
- React
- Vite
- TypeScript
- MapLibre
- OpenRouteService
- pytest
- Vitest

## Reference material

Assessment and FMCSA reference resources are stored in:

`docs/references/`
